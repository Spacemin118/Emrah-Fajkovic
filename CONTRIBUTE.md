# Contributing

Thanks for considering contributing to Elixir / Phoenix chat!

## Opening issues

If you find a bug, please feel free to [open an issue](https://github.com/Spacemin118/Emrah-Fajkovic/issues).

## Fixing bugs

Here’s a quick guide:

1. [Fork this repository](https://github.com/Spacemin118/Emrah-Fajkovic/fork) and then clone it locally:

```bash
git clone https://github.com/Spacemin118/Emrah-Fajkovic.git
```

2. Create a topic branch for your changes:

```bash
git checkout -b fix-for-that-thing
```

3. If everything looks good, push to your fork:

```bash
git push origin fix-for-that-thing
```

4. [Submit a pull request.](https://help.github.com/articles/creating-a-pull-request)
